docker_connect_https_fail

問題：在用 docker run 程式時，遇到無法連線 https 的問題，出現如下的 error

檢查的方式：curl -vv https://pypi.org

目前的解法

  • 用 ifconfig 確認 mtu 的值
  • 修改 docker config 的值，將 mtu 設定為跟網卡一樣
  • 這次遇到的 case，是將 mtu 設定為 1450
